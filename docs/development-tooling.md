# Development Tooling Baseline

This document lists the baseline development tools used by the project.

 # Development Tooling

 Recommended development tooling and baseline decisions:

 - **Python 3.11**
 - Use `venv` for virtual environments and `pip` for package management.
 - Test runner: `unittest` / `pytest` for unit and integration tests.
 - Code quality: `ruff` for linting and `black` for formatting.
 - Recommended editor: VS Code with Python extension.

 These baseline choices support consistent developer experiences across the project.
