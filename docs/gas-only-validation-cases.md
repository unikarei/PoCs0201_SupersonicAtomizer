# Gas-Only Validation Cases

Required gas-only validation cases:

- **Constant-Area Gas-Only Case**
- **Converging/Diverging Geometry Sanity Case**

These are gas only tests intended to verify solver trends where droplet physics are not present.

These test cases run with **no droplet transport enabled** to validate gas-only solver trends.

- Validation outcomes for gas-only cases should be reported with a pass/warn/fail status for CI and human review.

- Each gas-only validation case should include a short qualitative trend description to guide reviewers (qualitative trend).
