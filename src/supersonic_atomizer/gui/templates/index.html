    破砕モデルの説明

    Weber critical（Weber 閾値モデル）

    局所 Weber 数 We = ρg urel² d / σ が critical_weber_number を超えたら破砕を発生させる最もシンプルなモデルです。
    破砕後の液滴径は親径に固定係数を掛けて決めます。

    • critical_weber_number：破砕判定の閾値 We（デフォルト: 12）
    • breakup_factor_mean：平均径の縮小係数（例: 0.5 → 半径 50%）
    • breakup_factor_max：最大径の縮小係数

    用途: 感度検討・基準ケース確認。パラメータが直感的で分かりやすい。

    Bag-stripping（Pilch & Erdman 1987）

    Weber 数に応じて5つのレジームを自動判別します。 破砕後径は局所流条件から「安定 Weber 平衡径」として計算されます：
    dchild = Wecrit · σ / (ρg urel²)

    We 範囲レジーム
    We < 12安定（破砕なし）
    12 ≤ We < 18振動破砕
    18 ≤ We < 45バッグ破砕
    45 ≤ We < 351剥離・膜薄化
    We ≥ 351壊滅的破砕
    • critical_weber_number：安定径計算に使う Wecrit（デフォルト: 12）

    用途: 流れ場に応じて子径が自動変化するため Weber critical より物理的に整合的。

    KH-RT hybrid（Reitz 1987 / Beale & Reitz 1999）

    KH (Kelvin-Helmholtz) 不安定性による連続剥離と、 RT (Rayleigh-Taylor) 不安定性による壊滅的破砕を組み合わせたモデルです。 液体物性・ガス物性・スリップ速度から破砕後径を自動計算します。

    • khrt_B0：KH 波長係数（デフォルト: 0.61）
    • khrt_B1：KH 破砕時間係数（デフォルト: 40）
    • khrt_Crt：RT 波長係数（デフォルト: 0.1）
    • liquid_density：液体密度 [kg/m³]（デフォルト: 998.2、水 20°C）
    • liquid_viscosity：液体粘度 [Pa·s]（デフォルト: 1.002×10⁻³）

    用途: 高速噴霧での液柱・液滴の1次/2次破砕を物理的に表現したいとき。
